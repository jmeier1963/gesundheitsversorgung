# Interne Arbeitsdaten

Dieser Ordner ist fuer nicht-oeffentliche Arbeitsdaten vorgesehen, zum Beispiel Interviewnotizen, Fokusgruppenmaterial, Kontaktdaten, interne Bewertungen und nicht freigegebene Recherchezwischenstaende.

Der Ordner wird durch `.gitignore` vom Repository und damit auch von GitHub Pages ausgeschlossen. Nur diese README wird versioniert, damit die Trennung zwischen oeffentlichen und internen Daten sichtbar bleibt.
