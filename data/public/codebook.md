# Codebook: Öffentlicher Katalog

Der Datensatz `approaches.csv` enthaelt ausschliesslich öffentlich freigegebene, nicht-personenbezogene Informationen zu Praxisansätzen.

| Feld | Beschreibung |
|---|---|
| `id` | Stabile eindeutige Kennung, z. B. `de-001` oder `eu-nl-001`. |
| `name` | Name des Ansatzes oder Projekts. |
| `ort_region` | Gemeinde, Landkreis, Region oder Programmkulisse. |
| `bundesland_land` | Bundesland in Deutschland oder europäisches Vergleichsland. |
| `raumtyp` | Räumlicher Kontext, z. B. ländlich, strukturschwach, kleinstädtisch. |
| `zielgruppe` | Primaere Zielgruppe des Ansatzes. |
| `versorgungsthema` | Inhaltlicher Versorgungsschwerpunkt. |
| `massnahmentyp` | Typ des Ansatzes, z. B. mobile Versorgung, Gesundheitskiosk, Netzwerk. |
| `traeger_akteure` | Nicht-personenbezogene Träger- und Akteursgruppen. |
| `umsetzungsstand` | Status, z. B. Kandidat, in Umsetzung, abgeschlossen, skaliert. |
| `evidenz_quellenlage` | Kurzbewertung der Quellen- oder Evidenzlage. |
| `kriterienbewertung` | Bewertung entlang der Projektkriterien: `niedrig`, `mittel`, `hoch`, `zu prüfen`. |
| `kurzbeschreibung` | Kurzer öffentlicher Beschreibungstext. |
| `links` | Öffentliche Quellen-URL, falls vorhanden. |
| `latitude` | Breitengrad für die Kartenansicht. |
| `longitude` | Laengengrad für die Kartenansicht. |
| `veroeffentlichungsstatus` | Muss für die Website `public` sein. |

Nicht erlaubt sind Kontaktdaten, Interviewzitate ohne Freigabe, interne Bewertungen, personenbezogene Notizen oder sensible Rohdaten.
