# Codebook: Oeffentlicher Katalog

Der Datensatz `approaches.csv` enthaelt ausschliesslich oeffentlich freigegebene, nicht-personenbezogene Informationen zu Praxisansaetzen.

| Feld | Beschreibung |
|---|---|
| `id` | Stabile eindeutige Kennung, z. B. `de-001` oder `eu-nl-001`. |
| `name` | Name des Ansatzes oder Projekts. |
| `ort_region` | Gemeinde, Landkreis, Region oder Programmkulisse. |
| `bundesland_land` | Bundesland in Deutschland oder europaeisches Vergleichsland. |
| `raumtyp` | Raeumlicher Kontext, z. B. laendlich, strukturschwach, kleinstaedtisch. |
| `zielgruppe` | Primaere Zielgruppe des Ansatzes. |
| `versorgungsthema` | Inhaltlicher Versorgungsschwerpunkt. |
| `massnahmentyp` | Typ des Ansatzes, z. B. mobile Versorgung, Gesundheitskiosk, Netzwerk. |
| `traeger_akteure` | Nicht-personenbezogene Traeger- und Akteursgruppen. |
| `umsetzungsstand` | Status, z. B. Kandidat, in Umsetzung, abgeschlossen, skaliert. |
| `evidenz_quellenlage` | Kurzbewertung der Quellen- oder Evidenzlage. |
| `kriterienbewertung` | Bewertung entlang der Projektkriterien: `niedrig`, `mittel`, `hoch`, `zu pruefen`. |
| `kurzbeschreibung` | Kurzer oeffentlicher Beschreibungstext. |
| `links` | Oeffentliche Quellen-URL, falls vorhanden. |
| `veroeffentlichungsstatus` | Muss fuer die Website `public` sein. |

Nicht erlaubt sind Kontaktdaten, Interviewzitate ohne Freigabe, interne Bewertungen, personenbezogene Notizen oder sensible Rohdaten.
